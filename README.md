# 🌱 Carbon Footprint Tracker

A web-based Carbon Footprint Tracking System developed using Django to help monitor and manage industrial carbon emissions. The system allows factory owners to track energy usage, manage assets, and analyze CO₂ emissions, while admins can monitor reports, factories, and environmental impact data.

---

## 🚀 Features

### 👨‍💼 Admin Module
- Secure admin login
- Manage factory accounts
- View all factory emissions
- Generate weekly/monthly/yearly reports
- Monitor compliance and sustainability metrics
- Send alerts and notifications

### 🏭 Factory Owner Module
- Factory registration and login
- Add and manage factory assets
- Track electricity and fuel usage
- View calculated carbon emissions
- Access reports and dashboards
- Manage factory profile

## ▶️ How to Run the Project

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Sreelakshmi1621/Carbon-Footprint-Tracker.git
```

### 2️⃣ Navigate to Project

```bash
cd Carbon-Footprint-Tracker
```

### 3️⃣ Install Dependencies

```bash
pip install django
```

### 4️⃣ Run Server

```bash
python manage.py runserver
```

### 5️⃣ Open Browser

```text
http://127.0.0.1:8000/
```
